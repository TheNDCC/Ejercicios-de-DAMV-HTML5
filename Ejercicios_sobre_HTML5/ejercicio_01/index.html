¿Qué es HTML5?

HTML5 es la quinta versión del lenguaje de marcado estándar utilizado para crear y diseñar páginas web. Introduce nuevas características y mejoras que permiten una experiencia web más dinámica y multimedia.

Estructura básica de una página HTML5

Elementos

Los elementos HTML
son las partes constituyentes de una página web, como encabezados, párrafos, imágenes, enlaces, formularios, entre otros. Estos elementos se utilizan para estructurar y organizar el contenido de una página web.

Atributos

Los atributos HTML proporcionan información adicional sobre los elementos y les permiten interactuar con otros elementos y scripts. Los atributos pueden especificar características como el color, el tamaño, el enlace a otra página, entre otros.
