¿Qué es CSS3?

Publicado el 15 de octubre de 2012 a las 12h por Autor 1

CSS3 es la última versión del lenguaje de estilos CSS, y representa una evolución significativa con respecto a las versiones anteriores. Fue lanzado en 1999 y ha ido actualizándose a lo largo de los años para incluir nuevas funcionalidades y mejoras en la presentación de las páginas web.

Selectores, declaraciones, propiedades y valores de un CSS.

Publicado el 15 de octubre de 2012 a las 12h por Autor 1

Selectores CSS

  • Los selectores se utilizan para delimitar los elementos HTML a los que queremos aplicar estilo.
  • Un selector CSS es la primera parte de una regla CSS. Es un patrón de elementos y otros términos que indican al navegador qué elementos HTML se seleccionan para aplicarles una regla que incluye los valores de las propiedades CSS.
  • Por ejemplo, puedes seleccionar un elemento específico como h1 o una clase como .special.
  • Los selectores se definen en la especificación de selectores de CSS correspondiente y deben ser admitidos por los navegadores para funcionar correctamente. La mayoría de los selectores se encuentran en la especificación de nivel 3 de selectores, que es ampliamente compatible con los navegadores.

Declaraciones CSS

  • Una declaración de CSS es un bloque que especifica el conjunto de estilos que se añadirán a un elemento HTML.
  • Su estructura contiene lo siguiente:
    • Selector: Define el elemento o conjunto de elementos a los cuales se añadirán los estilos
    • Propiedad: Es el nombre del estilo de CSS.
    • Valor: Es el valor que tomará la propiedad2.

Propiedades y Valores

  • Las propiedades representan las características que deseas definir. Algunos ejemplos incluyen:
    • color: Define el color del texto.
    • font-size: Establece el tamaño de la fuente.
    • background-color: Define el color de fondo.
  • Los valores son los valores específicos que toma cada propiedad. Por ejemplo:
    • color: blue; establece el color del texto en azul.
    • font-size: 16px; define el tamaño de la fuente como 16 píxeles.
    • background-color: #f0f0f0; establece el color de fondo como gris claro

Formas de añadir CSS a un HTML (Externo, Interno y en línea)

  • CSS Externo:
    • En este enfoque, defines una hoja de estilo CSS en un archivo separado del documento HTML.
    • Utilizas el elemento en la sección del HTML para enlazar la hoja de estilo externa.
  • CSS Interno:
    • En este método, el código CSS está incluido directamente dentro del documento HTML.
    • Utilizas la etiqueta _Stile en la sección head_ para definir las reglas de estilo.
  • CSS en Línea:
    • Aquí aplicas estilos directamente a un elemento HTML utilizando el atributo style.
