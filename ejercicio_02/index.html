<header>

Define la cabecera de un documento o de una sección.

<footer>

Define el pie de página de un documento o de una sección.

<nav>

Define un conjunto de enlaces de navegación.

<article>

Define un contenido independiente y autónomo, como un post de blog o una noticia.

<section>

Define una sección temática en un documento.

<aside>

Define contenido que es tangencial al contenido principal, como barras laterales o anuncios.

<main>

Define el contenido principal de un documento.

<figure>

Define contenido independiente, como imágenes, gráficos, diagramas, etc., y opcionalmente una leyenda <figcaption>.

<figcaption>

Define la leyenda de una figura.

<details>

Define un control de detalles interactivo que puede ser abierto o cerrado por el usuario para revelar o esconder más información.

<summary>

Define un título visible para el contenido de un elemento <details>.

<time>

Define una fecha o hora específica.

<mark>

Resalta o marca un fragmento de texto dentro de un documento.

<progress>

Representa el progreso de una tarea.

<meter>

Representa un valor escalar dentro de un rango conocido o desconocido.
